# Physical AI & Humanoid Robotics

Welcome to **Physical AI & Humanoid Robotics**: A Practical Guide to ROS 2, Simulation, and Vision-Language-Action Systems.

## What You'll Learn

This book teaches you how to build intelligent humanoid robots through four comprehensive modules:

### Module 1: Robotic Nervous System
- ROS 2 foundations
- URDF robot descriptions
- Transform trees (TF2)
- Python integration with rclpy

### Module 2: Digital Twin
- Physics simulation with Gazebo
- Sensor simulation (cameras, LiDAR, IMU)
- Unity integration for photorealistic rendering

### Module 3: AI-Robot Brain
- NVIDIA Isaac Sim
- Computer vision and object detection
- Visual SLAM and localization
- Navigation with Nav2

### Module 4: Vision-Language-Action
- Voice control with Whisper
- LLM-based task planning
- Multimodal VLA systems
- Capstone: Autonomous humanoid butler

## Prerequisites

- Basic Python programming
- Familiarity with Linux command line
- No robotics experience required!
- Hardware: 16GB+ RAM, GPU recommended for Module 3-4

## Getting Started

Ready to dive in? Start with Module 1: Robotic Nervous System.
