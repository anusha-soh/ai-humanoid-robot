# Prerequisites

This page outlines everything you need to successfully complete the Physical AI & Humanoid Robotics book.

## Software Requirements

### Operating System
- **Ubuntu 22.04 LTS** (recommended)
- Windows 11 with WSL2 (Ubuntu 22.04)
- macOS 12+ (limited Isaac Sim support)

### Core Tools
- **ROS 2 Humble** (installation guide in Chapter 1)
- **Python 3.10+**
- **Node.js 18+** (for web-based visualizations)
- **Git** for version control

### Simulation Software
- **Gazebo Classic 11** (Modules 1-2)
- **NVIDIA Isaac Sim 2023.1+** (Modules 3-4, GPU required)

## Hardware Requirements

### Minimum
- **CPU**: 4 cores, 2.5GHz+
- **RAM**: 16GB
- **Storage**: 50GB free space
- **GPU**: Integrated graphics (Modules 1-2 only)

### Recommended
- **CPU**: 8+ cores, 3.0GHz+
- **RAM**: 32GB
- **Storage**: 100GB SSD
- **GPU**: NVIDIA RTX 2060 or better (for Modules 3-4)

## Knowledge Prerequisites

### Required
- **Python Basics**: Variables, functions, loops, classes
- **Command Line**: Navigate directories, run commands
- **Text Editor**: VS Code, Vim, or similar

### Helpful (Not Required)
- Linear algebra (vectors, matrices)
- Basic robotics concepts
- ROS 1 experience

## Installation Checklist

Detailed installation steps are provided in Chapter 1. Quick checklist:

- [ ] Ubuntu 22.04 installed (native or WSL2)
- [ ] ROS 2 Humble installed
- [ ] Python 3.10+ with pip
- [ ] Gazebo Classic 11
- [ ] Git configured
- [ ] Code editor installed

## GPU Setup (Modules 3-4)

If you have an NVIDIA GPU:
- [ ] NVIDIA drivers 525+
- [ ] CUDA 11.8+
- [ ] cuDNN 8.6+

**Don't have a GPU?** No problem! Gazebo fallback exercises are provided for all Module 3-4 content.

## Estimated Time Commitment

- **Module 1**: 2-3 weeks (8-12 hours/week)
- **Module 2**: 2-3 weeks
- **Module 3**: 3-4 weeks
- **Module 4**: 3-4 weeks
- **Total**: 10-14 weeks (2-4 hours per chapter)
