# Glossary

Key terms and concepts used throughout the book.

## A

**Action Space**: The set of all possible actions a robot can take.

## D

**DDS (Data Distribution Service)**: The middleware protocol ROS 2 uses for communication.

## G

**Gazebo**: Open-source 3D robot simulator with physics engine.

## I

**Isaac Sim**: NVIDIA's photorealistic robot simulation platform with GPU acceleration.

## L

**LLM (Large Language Model)**: AI models trained on vast text data for natural language understanding and generation.

## N

**Nav2**: ROS 2 navigation stack for autonomous robot navigation.

## R

**ROS 2 (Robot Operating System 2)**: Open-source middleware for robot software development.

**rclpy**: Python client library for ROS 2.

## S

**SLAM (Simultaneous Localization and Mapping)**: Technique for building maps while tracking robot position.

## T

**TF2 (Transform Library 2)**: ROS 2 library for tracking coordinate frame transformations.

## U

**URDF (Unified Robot Description Format)**: XML format for describing robot kinematics and dynamics.

## V

**VLA (Vision-Language-Action)**: Multimodal AI systems that combine visual perception, language understanding, and action execution.

**Visual SLAM**: SLAM using camera data instead of LiDAR.

## W

**Whisper**: OpenAI's speech recognition model.
